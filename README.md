 Game Scoreboard Management System (DBMS)

A Python-based desktop application that interacts with a MySQL database to manage player statistics.

Features
Persistent Storage: Uses MySQL for robust data management.
Real-time Leaderboard: Automatically sorts players by highest score.
Client-Server Architecture: Connects Python (Client) to XAMPP/MySQL (Server).

Tech Stack
Language: Python 3.x
Database: MySQL (via XAMPP)
Library: `mysql-connector-python`

 Setup Instructions
1. Install XAMPP and start Apache & MySQL.
2. Create a database named `game_db` in phpMyAdmin.
3. Install dependencies: `pip install mysql-connector-python`.
4. Run the application: `python main.py`.
